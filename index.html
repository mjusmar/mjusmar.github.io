02 — Research

Master's Thesis

Object-Level Semantic Gaussian Mapping

A modular pipeline for semantically-enriched 3D scene reconstruction using Gaussian Splatting, temporal mask propagation, and open-vocabulary labeling.

SplaTAM SAM2 CLIP Gaussian Splatting RGB-D Semantic SLAM
Ongoing · 2024–2026

Pipeline Stages

01
2D Segmentation

SAM2 video predictor for temporally consistent object masks across RGB-D frames.

02
2D → 3D Projection

Back-project masked depth pixels into world coordinates preserving object IDs.

03
3D Extraction & Cleaning

Voxel downsampling, statistical outlier removal, and DBSCAN clustering per object.

04
Semantic Labeling

Multi-view CLIP feature aggregation from top-k object crops for open-vocabulary classification.

05
Gaussian Association

KD-tree radius search links semantic labels to Gaussian primitives for object-aware map visualization.

Key Results

0.85 Mean Reproj. IoU
0.93 Depth Consistency
>0.95 Gaussian Coverage
0.63 Temporal IoU (SAM2)
